快速启动



其他载荷

点击这里





结果：
  1. 将 Switch 置于 RCM 模式，并将其连接到你的设备。
  2. 选择示例载荷或上传一个。
  3. 点击"发送载荷！"
  4. 在出现的确认界面上，选择"APX"并点击确认。
  5. 如果一切顺利，载荷将会启动！


修改自 WebFG， 该项目移植自 fusee-launcher。感谢 atlas44 提供的 java 移植版本，感谢 rajkosto 提供的众多载荷， 感谢 ktemkin 和 ReSwitched 开发的 Fusée Gelée 以及其他众多贡献！
  • 这个程序测试不太充分。我只是写写看。如果出现任何问题，我不负责！
  • 由于 Chrome 的 WebUSB 实现限制（可能还有其他原因），此程序在 Windows 上无法运行！
  • 由于其他浏览器不支持 WebUSB，此程序目前仅支持 Chrome 浏览器。
  • 在 Linux 系统上，你可能会遇到访问权限被拒绝的错误！如果发生这种情况， 你可以尝试在此路径创建文件： /etc/udev/rules.d/50-switch.rules
    文件内容如下：
    SUBSYSTEM=="usb", ATTR{idVendor}=="0955", MODE="0664", GROUP="plugdev"
  • 此程序已在 Linux、OSX、Android（未 root）和 Chromebook 上测试通过。具体效果可能因设备而异。
